# AI-Powered Fake News Detection System

An end-to-end AI-powered Fake News Detection system built using NLP and Machine Learning. The model is deployed as a FastAPI service and integrated with WhatsApp using Twilio webhooks.

---

## 🚀 Features
- Fake vs Real news classification
- TF-IDF text vectorization
- Naive Bayes classifier
- REST API using FastAPI
- WhatsApp integration using Twilio
- Public exposure using ngrok

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- FastAPI
- Twilio API
- ngrok
- Git & GitHub
